# API Documentation

## Overview

Test API for e-commerce platform

## Base URL

```
http://localhost:8000/api/v1
```

## Authentication

This API uses JWT Bearer token authentication. Include the token in the Authorization header:

```
Authorization: Bearer <token>
```
